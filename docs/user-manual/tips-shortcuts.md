# Tips & Keyboard Shortcuts

Maximize your productivity with PM Desktop using these power-user features, keyboard shortcuts, and tips.

---

## Keyboard Shortcuts

### Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open application selector (Notes) |
| `Escape` | Close modal, panel, or menu |
| `Tab` | Navigate between fields |
| `Enter` | Confirm action, submit form |

### Navigation

| Shortcut | Action |
|----------|--------|
| `Arrow Keys` | Navigate lists and menus |
| `Enter` | Select highlighted item |
| `Space` | Toggle checkbox |
| `Backspace` | Go back (in some contexts) |

### Text Editing

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + B` | Bold text |
| `Cmd/Ctrl + I` | Italic text |
| `Cmd/Ctrl + U` | Underline text |
| `Cmd/Ctrl + K` | Insert/edit link |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |
| `Cmd/Ctrl + A` | Select all |
| `Cmd/Ctrl + C` | Copy |
| `Cmd/Ctrl + V` | Paste |
| `Cmd/Ctrl + X` | Cut |

### Task Editor

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Enter` | Submit comment |
| `@` | Start @mention |
| `#` | Tag reference (coming soon) |
| ``` ` ``` | Inline code |

### Kanban Board

| Shortcut | Action |
|----------|--------|
| `Arrow Keys` | Navigate task cards |
| `Enter` | Open selected task |
| `Escape` | Close task detail panel |
| `N` | New task (when no input focused) |

## Productivity Tips

### Quick Actions

1. **Dashboard shortcuts**: Use Quick Action buttons for fast navigation
2. **Breadcrumb navigation**: Click any level to jump back
3. **Double-click**: Edit items inline where supported

### Efficient Task Management

1. **Bulk operations**: Select multiple tasks with Ctrl/Cmd + Click
2. **Keyboard navigation**: Arrow through tasks without mouse
3. **Quick create**: Type and press Enter immediately

### Comment Efficiency

1. **@Mention dropdown**: Start typing name after @ for quick selection
2. **Paste images**: Directly paste screenshots from clipboard
3. **Markdown shortcuts**: Use formatting marks instead of toolbar

### Note-Taking Speed

1. **Tab to save time**: Open multiple notes, switch with tabs
2. **Keyboard formatting**: Faster than clicking toolbar buttons
3. **Application switcher**: `Cmd/Ctrl + K` for instant switching

## Search Tips

### Effective Searching

1. **Be specific**: Use unique terms for better results
2. **Use filters**: Combine search with filters
3. **Search by key**: Task keys (e.g., "SP1-42") for direct access

### Search Patterns

| Pattern | Finds |
|---------|-------|
| `bug` | All items containing "bug" |
| `login fix` | Items with both "login" and "fix" |
| `SP1-*` | All tasks in project SP1 |

## Interface Tips

### Sidebar

1. **Collapse sidebar**: Gain more space for content
2. **Hover for tooltips**: See full names on truncated items
3. **Right-click menus**: Additional options often available

### Kanban Board

1. **Scroll horizontally**: Mouse wheel or trackpad gesture
2. **Collapse columns**: Focus on active work
3. **Drag anywhere**: Drop zone highlights help targeting

### Modals and Panels

1. **Click outside**: Often closes the modal
2. **Tab through fields**: Efficient form completion
3. **Enter to submit**: Quick confirmation

### Theme

1. **System preference**: Follows your OS light/dark setting
2. **Manual toggle**: Override in title bar
3. **Consistent experience**: Theme applies everywhere

## Collaboration Tips

### Effective @Mentions

| Do | Don't |
|----|-------|
| Mention with clear ask | Mention without context |
| Use for action items | Over-mention the same person |
| Include relevant context | Mention in every comment |

2. Coordinate edits when multiple editors
3. Use comments to claim work

### Clear Communication

1. Be specific in task titles

4. Reference related tasks
