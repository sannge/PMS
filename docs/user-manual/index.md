## Quick Overview

### Application Hierarchy

PM Desktop organizes work in a three-level hierarchy:

```
Applications (Top Level)
    └── Projects (Mid Level)
            └── Tasks (Work Items)
```

- **Applications**: Top-level containers representing major initiatives, products, or departments
- **Projects**: Groupings of related work within an application (e.g., sprints, features)
- **Tasks**: Individual work items with status, assignees, checklists, comments, and subtasks

### Key Features

| Feature | Description |
|---------|-------------|
| Kanban Board | Visual task management with drag-and-drop status updates and custom workflows |
| Real-time Collaboration | See team members' presence and get instant updates via WebSocket |
| Rich Comments | Formatted text with @mentions, threaded replies, and file attachments |
| Checklists | Track sub-tasks and requirements within tasks |
| Knowledge Base | Collaborative notes with real-time editing, folders, tags, full-text search, and document locking |
| Canvas Diagrams | Draw.io integration for creating and editing diagrams |
| Document Import | Batch import PDF, Word, PowerPoint, Excel, and Visio files into the knowledge base |
| AI Assistant (Blair) | Natural-language assistant that can search documents, create tasks, manage projects, and more |
| Role-based Access | Owner, Manager, Member, and Viewer permission levels |
| Dashboard | Project health charts, task distribution, and completion trends |
| Tags | Categorize and filter documents with custom tags |

### Navigation

The main sidebar provides quick access to all sections:

- **Dashboard**: Overview with statistics, charts, and quick actions
- **Applications**: Browse and manage all applications
- **Projects**: Direct access to project list
- **Notes**: Knowledge base with hierarchical documents, diagrams, and search
- **AI Assistant**: Chat with Blair to search your data, create tasks, or get answers
- **Settings**: Application preferences

## System Requirements

- **Operating System**: Windows 10/11, macOS 10.15+, or Linux (Ubuntu 20.04+)
- **Memory**: 4GB RAM minimum, 8GB recommended
- **Display**: 1280x720 minimum resolution
- **Network**: Stable internet connection for real-time features
