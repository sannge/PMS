# Getting Started

This guide covers initial setup, account creation, and first steps with PM Desktop.

---

## Installation

### Windows

1. Download the PM Desktop installer (`.exe`) from your organization's software portal
2. Run the installer and follow the on-screen prompts
3. Launch PM Desktop from the Start menu or desktop shortcut

### macOS

1. Download the PM Desktop disk image (`.dmg`)
2. Open the disk image and drag PM Desktop to your Applications folder
3. Launch PM Desktop from Applications or Spotlight

### Linux

1. Download the AppImage or `.deb` package
2. For AppImage: Make it executable with `chmod +x` and run directly
3. For `.deb`: Install with `sudo dpkg -i pm-desktop.deb`

---

## Creating an Account

If you don't have an account yet:

1. Launch PM Desktop
2. Click **Register** on the login screen
3. Fill in the registration form:
   - **Email**: Your work email address (required)
   - **Password**: Minimum 8 characters (required)
   - **Confirm Password**: Re-enter your password
   - **Display Name**: How you'll appear to teammates (optional)
4. Click **Create Account**
5. Upon successful registration, you'll see a confirmation message
6. Click **Back to Login** to sign in

### Password Requirements

- Minimum 8 characters
- We recommend using a mix of letters, numbers, and symbols

---

## Logging In

1. Launch PM Desktop
2. Enter your **Email** and **Password**
3. Click **Sign In**

### Password Visibility

Click the eye icon in the password field to toggle visibility.

### Staying Signed In

Your session persists until you explicitly log out or your session token expires.

## First-Time Setup

After logging in for the first time:

### 1. Explore the Dashboard

The Dashboard provides an overview of your work:

- **Quick Stats**: Applications, Projects, Active Tasks, and Completed counts
- **Quick Actions**: Buttons to create new applications or view projects
- **Recent Activity**: Updates from your team (coming soon)

### 2. Set Your Theme

Click the theme toggle in the title bar to switch between:

- **Light Mode**: Bright interface for well-lit environments
- **Dark Mode**: Reduced eye strain for low-light conditions
- **System**: Follows your operating system preference

### 3. Join or Create an Application

- If invited: Check your notifications for pending invitations
- If starting fresh: Click **New Application** from the Dashboard or Applications page

## Understanding the Interface

### Title Bar

Located at the top of the window:

- **Application name**: PM Desktop logo and title
- **Theme toggle**: Switch between light/dark modes
- **Notification bell**: Shows unread notification count
- **User menu**: Access profile and logout options

### Sidebar Navigation

The left sidebar provides primary navigation:

| Icon | Section | Description |
|------|---------|-------------|
| Grid | Dashboard | Overview and quick stats |
| Squares | Applications | List all your applications |
| Folder | Projects | Direct project access |
| File | Notes | Knowledge base and documentation |
| Gear | Settings | Preferences (coming soon) |

**Tip**: Click the collapse button to minimize the sidebar and gain more screen space.

### Main Content Area

The central area displays the current section's content. This adapts based on your selection:

- Card grids for applications and projects
- Kanban board for project tasks
- Multi-tab editor for notes

## Your First Application

Let's create your first application to start managing work:

1. Navigate to **Applications** in the sidebar
2. Click the **New** button in the top-right
3. Enter an **Application Name** (e.g., "Product Development")
4. Optionally add a **Description**
5. Click **Create**

You're now the Owner of this application and can:

- Create projects within it
- Invite team members
- Manage all content

See [Applications](./applications.md) for detailed application management.

---

## Your First Project

With an application created:

1. Click on your application to open it
2. Click **New Project** or the **+** button
3. Fill in the project form:
   - **Name**: Project display name (e.g., "Sprint 1")
   - **Key**: Short unique identifier (e.g., "SP1")
   - **Description**: Optional project description
4. Click **Create**

The project now appears in your application. Click it to access the Kanban board.

See [Projects](./projects.md) for detailed project management.

---

## Your First Task

Within a project's Kanban board:

1. Find the **To Do** column (or your first status column)
2. Click the **+** button or **Add Task** link
3. Enter a task title (e.g., "Set up development environment")
4. Press Enter or click Create

Your task appears as a card on the board. Click it to:

- Add a description
- Assign team members
- Set priority and type
- Add checklists
- Leave comments

See [Tasks & Kanban Board](./tasks.md) for complete task management.

## Logging Out

To sign out of PM Desktop:

1. Click your user avatar/name in the title bar
2. Select **Logout** from the dropdown menu
3. You'll be returned to the login screen

Your session data is cleared, but any work is saved to the server.

## Troubleshooting

### Can't Log In

- Verify your email and password are correct
- Check your Caps Lock key
- Try resetting your password if available
- Contact your administrator if issues persist

### Connection Issues

- Check your internet connection
- The application will automatically reconnect when network is restored
- A connection status indicator appears when offline

### Application Won't Launch

- Ensure your system meets minimum requirements
- Try reinstalling the application
- Check for conflicting software or firewall rules

For additional help, contact your system administrator.
